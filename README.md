Convertify 🚀
A modern multi-unit converter web application built using ReactJS and Tailwind CSS.
Users can convert values between different units like:


Celsius ↔ Fahrenheit


Kilometers ↔ Miles


The project was created to learn and implement core React concepts such as:


Components


Props


State management using useState


Dynamic rendering


Reusable UI components


Event handling


Controlled components



📌 Features
✅ Temperature Conversion
✅ Length Conversion
✅ Dynamic Sidebar Categories
✅ Reusable Dropdown Components
✅ Instant Result Updates
✅ Responsive Modern UI
✅ Tailwind CSS Styling
✅ Component-Based Architecture

🧠 React Concepts Used
useState
Used for managing:


input value


selected category


selected units



Props
Props are used for communication between parent and child components.

Component Reusability
Reusable components like:


UnitSelect


InputField


ResultDisplay


help keep the code modular and maintainable.

Dynamic Rendering
map() and Object.keys() are used for dynamically generating:


sidebar buttons


dropdown options



Controlled Components
The input field and dropdown values are controlled using React state.
